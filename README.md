# 🐉 BDH Sparse Brain Visualizer

This project demonstrates a fundamentally new property in post-transformer architectures:

👉 Sparse neural computation that is directly interpretable in real time.

Unlike transformers where ~100% of neurons are always active, BDH naturally produces sparse activations, enabling:

- Direct neuron-level interpretability
- Constant memory independent of sequence length
- A graph-like structure of computation

This demo allows you to SEE how a language model thinks.
